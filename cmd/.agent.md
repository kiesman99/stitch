# CLI Commands

This directory contains Cobra CLI commands for the stitch application.

## Structure

- `root.go`: Root command with global configuration and Viper setup
- `stitch.go`: Main stitch command for downloading and stitching tiles

## Adding New Commands

To add new commands (like a future REST server command):

1. Create a new file in this directory (e.g., `server.go`)
2. Define the command using Cobra patterns
3. Add it to the root command in the `init()` function
4. Use Viper for configuration binding

## Configuration

Commands use Viper for configuration management, supporting:
- Command line flags
- Configuration files (YAML)
- Environment variables

The configuration hierarchy (highest to lowest priority):
1. Command line flags
2. Environment variables
3. Configuration file
4. Default values
